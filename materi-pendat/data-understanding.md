# Data Understanding

Tahap **Data Understanding** berfokus pada mengenali karakteristik data
yang dimiliki. Pada bagian ini kita mengecek apakah data:
- relevan dengan tujuan bisnis,
- cukup banyak,
- dan kualitasnya layak untuk dianalisis lebih jauh.

## Deskripsi data
- Menjelaskan **sumber data** (misalnya: sistem akademik, hasil survei,
  file CSV, database, API, dan lain‑lain).
- Menuliskan jumlah **baris** (record) dan **kolom** (fitur) yang tersedia.
- Merangkum tipe data tiap kolom (numerik, kategorikal, tanggal, teks, dsb.)
  beserta contoh isinya, supaya pembaca punya gambaran konkret.

## Eksplorasi awal (EDA)
- Menghitung **statistik deskriptif**: rata‑rata, median, minimum, maksimum,
  standar deviasi, dan sebaran nilai untuk variabel penting.
- Mengecek **missing value** di setiap kolom, pola kemunculannya,
  dan kemungkinan penyebabnya (misalnya kolom yang jarang diisi).
- Mengidentifikasi **outlier/anomali** menggunakan grafik seperti boxplot,
  histogram, atau scatter plot.
- Menuliskan **insight awal**, misalnya:
  - distribusi nilai cenderung rendah/tinggi,
  - ada fitur yang hampir selalu kosong,
  - atau ada korelasi kuat antara dua variabel tertentu.
